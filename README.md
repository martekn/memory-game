# Zelda memory game

## Getting Started

### Installing

#### 1. Clone the repo:

```bash
git clone git@github.com:martekn/memory-game.git
```

#### 2. Install the dependencies:

```bash
npm install
```

### Running

#### Compile and Hot-Reload for Development

```bash
npm run dev
```

#### Compile and Minify for Production

```bash
npm run build
```
